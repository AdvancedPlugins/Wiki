# PlaceholderAPI Placeholders

-   `%advancedchallenges_top_<name/value/value_formatted/place>_<place/me>_<points/streak_daily/streak_playtime/streak_vote>%` - Returns leaderborad data (place in a first argument is a static text which means to return player's place. Place in the second argument is a number from which place you want data)
-   `%advancedchallenges_has_booster_<TARGET>_<TYPE>_<PERCENT>_<AFFECTS>%` - Returns `true` / `false` if player has certain booster. Instead of each argument you can use `ANY` to get any booster
-   `%advancedchallenges_points%` - Player points
-   `%advancedchallenges_points_formatted%` - Formatted result of player points
-   `%advancedchallenges_unclaimed_rewards%` - Number of unclaimed rewards
-   `%advancedchallenges_streak_<daily/playtime/vote>%` - Streak level
-   `%advancedchallenges_time_to_reroll_<category>%` - Time to reroll certain category
-   `%advancedchallenges_completed_challenges_<category>%` - Number of completed challenges (category is optional)
-   `%advancedchallenges_not_completed_challenges_<category>%` - Number of not completed challenges (category is optional)
-   `%advancedchallenges_available_challenges_<category>%` - Number of all available (completed and not) challenges (category is optional)
